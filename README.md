# ATRIUM_APP

Application statique (HTML/CSS/JS vanilla) d'apprentissage du latin (A1+/A2-) orientée production contrôlée.

## Principes implémentés
- 3 périodes
- 5 leçons par période (15 leçons)
- Chaque leçon : entraînement `/7`, production écrite guidée `/3`, total `/10`
- Chaque période : score `/50`, pourcentage, statut validée/non validée (validation à partir de 80 %)
- Persistance via `localStorage`
- Correction de la production avec normalisation des réponses

## Lancer en local
Ouvrir `index.html` directement, ou servir le dossier avec un serveur statique.
